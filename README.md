# Famicom.js

### JavaScript ファミコンエミュレータ

NES Emulator
<br><br><br><br>


  enter = START  
  shift = SELECT  
  z = B  
  a = A  

  Famicom のSaveに対応しました。rom file name　と紐ついているのでファイル名を変えると消えます。
